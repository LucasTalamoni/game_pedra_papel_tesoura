# Jogo Pedra, Papel e Tesoura

Este é um script simples em Python que implementa o clássico jogo "Pedra, Papel e Tesoura". Você joga contra o computador, que escolhe uma opção aleatoriamente.

## Funcionalidades

- Escolha aleatória da máquina utilizando a biblioteca `random`.
- Tratamento da entrada do usuário (remove espaços e ignora maiúsculas/minúsculas).
- Lógica de comparação para determinar vitória, derrota ou empate.

## Pré-requisitos

Você precisa ter o Python instalado em sua máquina.

## Como executar

1. Abra o terminal.
2. Navegue até o diretório onde o arquivo `Game.py` está salvo.
3. Execute o comando:

```bash
python Game.py
```

4. Digite `pedra`, `papel` ou `tesoura` quando solicitado e veja se você venceu a máquina!
